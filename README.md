# KNN Bias-Variance Tradeoff Project

## Objective
To study how different K values affect overfitting and underfitting in KNN.

## Concepts Used
- KNN
- Bias Variance Tradeoff
- Feature Scaling
- Model Evaluation

## Dataset
Heart Disease Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit

## How to Run

### Install Libraries
pip install -r requirements.txt

### Run Model
python train_model.py

### Run Dashboard
streamlit run app.py
